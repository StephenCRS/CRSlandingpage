# Water Landing Page (starter)

This folder contains a simple static landing page you can edit and deploy.

How to update
- Edit water-landing-page/index.html and css/style.css.
- Replace the placeholder email and phone with your real contact details.
- Commit changes to GitHub; Netlify (or other hosts) can then publish the folder.

Deploying to Netlify (one-off upload)
- Zip the contents of this folder (make sure index.html is at the archive root) and drag to https://app.netlify.com/drop
